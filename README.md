# Nexus Admin Dashboard

A premium, modern React admin dashboard built with Vite, React, Lucide-React, and Recharts.

## 🚀 Getting Started

To run this project locally, follows these steps:

1. **Check Internet Connection**: Ensure you have an active internet connection to download dependencies.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **View in Browser**: Open the link provided in the terminal (usually `http://localhost:5173`).

## 📁 Key Components

- **App.jsx**: The heart of the dashboard including Navigation, Stats, and Charts.
- **index.css**: Premium design system with custom color variables and glassmorphism styles.
- **Main.jsx**: React entry point.

## 🎨 Features

- Dark mode premium aesthetic.
- Interactive Area Charts for revenue tracking.
- Statistical cards with trend indicators.
- Responsive Sidebar and Header.
- Clean, thoroughly commented code.
